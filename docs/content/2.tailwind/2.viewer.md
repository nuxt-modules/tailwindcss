# Viewer

Visualize your Tailwind configuration with the viewer.

---

Nuxt Tailwind exposes a `/_tailwind/` route in development, allowing you to quickly visualize your Tailwind configuration with easy copy-pasting (thanks to the awesome [tailwind-config-viewer](https://github.com/rogden/tailwind-config-viewer) package âœ¨).

The viewer is available starting from version  `v3.4.0` of `@nuxtjs/tailwindcss`.
